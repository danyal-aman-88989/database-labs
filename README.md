# Database Labs Repository

## Overview

This repository contains a collection of labs and exercises for learning and practicing database management and SQL queries. The primary focus of these labs is to provide hands-on experience with PostgreSQL, including database creation, schema design, querying, and troubleshooting.

## Contents

* **Lab 1: PostgreSQL Setup and Connection**

  * Steps for installing and setting up PostgreSQL.
  * Connecting to PostgreSQL using different tools (e.g., psql, DBeaver).
  * Configuration and troubleshooting steps.

* **Lab 2: Database Schema Design**

  * Creating and modifying database schemas.
  * Adding tables, defining relationships, and setting constraints.
  * Using SQL commands for schema creation and modifications.

* **Lab 3: Querying Databases**

  * Writing SQL queries for selecting, updating, and deleting data.
  * Using advanced SQL techniques like joins, subqueries, and aggregate functions.
  * Filtering, sorting, and grouping data effectively.

* **Lab 4: Data Manipulation and Integrity**

  * Inserting, updating, and deleting records in tables.
  * Understanding and enforcing data integrity through constraints.
  * Transactions and rollbacks.

* **Lab 5: Export and Backup**

  * Using `pg_dump` to export database schemas and data.
  * Restoring backups and managing database security.
  * Managing roles and permissions in PostgreSQL.

* **Lab 6: Troubleshooting PostgreSQL**

  * Resolving common PostgreSQL errors and connection issues.
  * Configuring authentication methods in `pg_hba.conf` and `postgresql.conf`.
  * Best practices for monitoring and maintaining PostgreSQL databases.

## Requirements

* PostgreSQL 12 or later (preferably version 16.11 for compatibility).
* A working Linux environment (Ubuntu or other Linux distros recommended).
* Tools like **psql** or **DBeaver** for database interaction.
* Basic understanding of SQL and database concepts.

## Setup Instructions

1. **Clone this repository** to your local machine:

   ```bash
   git clone https://github.com/yourusername/database-labs.git
   ```

2. **Navigate** to the repository directory:

   ```bash
   cd database-labs
   ```

3. Follow the instructions in each lab folder for setup and execution.
